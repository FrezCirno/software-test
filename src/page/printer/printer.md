# 打印机

## 打印机系统

**事件**

- （S）initialize事件
- （1）请求复印事件
- （2）复印结束事件
- （3）检查复印机状态事件
- （4）检查复印机状态结束事件
- （E）delete事件

**状态**

0. Initial状态

1. 空闲状态
2. 复印状态
3. 警报状态
4. End状态
5. Failure状态
